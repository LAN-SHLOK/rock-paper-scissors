# Rock Paper Scissors

A functional implementation of the classic Rock, Paper, Scissors game. This project demonstrates core programming logic, conditional statements, and user input handling in a lightweight format.

## Features
- Interactive gameplay against a computer opponent.
- Score tracking for multiple rounds.
- Input validation to ensure robust user experience.

## Usage
Run the main script and follow the on-screen prompts to input your choice (Rock, Paper, or Scissors). The system will randomly generate a choice for the computer and determine the winner of the round.
